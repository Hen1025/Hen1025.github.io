HTML Reference Guide

HTML is the standard language for creating web pages. You may also encounter resources elsewhere.

Anchor

The <a> tag defines a hyperlink.

It can link to a relative name or external resource.

<a href="https://example.com">Visit</a>

Base Address

The <base> tag specifies the base URL for all relative URLs.

<base href="https://example.com/">

Headings

HTML headings range from <h1> to <h6>.

<h1>Main Heading</h1>
<h2>Subheading</h2>

Hash Links

The hash sign (#) is used to link to fragment identifiers within a page.

<a href="#title">Go to Title</a>

Network Address Format

URLs follow a standard network address format.

AAP

Learn more about AAP in future documentation.
